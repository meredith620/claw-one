# 产品形态详细设计

## 1. Claw Desktop（免费软件版）

### 定位

面向个人 Geek、分析师、早期用户的免费软件，降低试用门槛。

### 技术栈

| 层级 | 技术选型 |
|------|---------|
| **后端** | Claw Manager (Node.js / Rust 待决策) |
| **前端** | Web UI (React/Vue 待决策) |
| **封装** | Tauri (Rust + Webview) |
| **打包** | Windows (.msi) / macOS (.dmg) / Linux (.AppImage/.deb) |

### 功能清单

| 模块 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| **安装向导** | 环境检测 | P0 | 检测 Node.js、端口占用等 |
| | 自动安装依赖 | P0 | 自动下载安装 Node.js、OpenClaw |
| | 初始化配置 | P0 | 引导式配置模型和渠道 |
| **配置管理** | 可视化表单 | P0 | 表单替代 JSON 编辑 |
| | JSON Schema 校验 | P0 | 实时校验配置合法性 |
| | 事务性应用 | P0 | backup→apply→healthcheck |
| | 配置快照 | P0 | 自动保存历史版本 |
| **运行时管理** | 启动/停止/重启 | P0 | 控制 OpenClaw 生命周期 |
| | 日志查看 | P0 | 实时日志流 |
| | 运行时切换 | P1 | OpenClaw/PicoClaw/Nanobot |
| **技能市场** | 技能浏览 | P1 | 图形化展示可用技能 |
| | 一键安装 | P1 | 简化技能安装流程 |
| | 安全提示 | P1 | 安装前权限说明 |
| **快照回滚** | 历史版本 | P0 | 查看最近 10 个配置版本 |
| | 一键回滚 | P0 | 快速恢复到之前版本 |
| | 导出/导入 | P1 | 配置备份迁移 |
| **Safe Mode** | 自动检测 | P0 | Gateway 崩溃自动进入 |
| | 错误诊断 | P0 | 人话解释错误原因 |
| | 恢复操作 | P0 | 回滚/重置操作 |
| **远程诊断** | 诊断包生成 | P1 | 一键导出诊断信息 |
| | 加密上传 | P1 | 安全发送给技术支持 |
| **更新管理** | 版本检查 | P1 | 检测新版本 |
| | 一键更新 | P1 | 自动备份后更新 |
| | 版本回滚 | P1 | 回退到旧版本 |

### 安全隔离

**默认：Level 1 用户沙盒**

- 创建专用用户 `clawuser`
- 受限文件访问（只能访问 `~/.openclaw/` 和工作目录）
- 进程命名空间隔离
- cgroup 资源限制（CPU 20%，内存 4GB）

**可选：Level 0 裸机运行**

- 高级用户可关闭沙盒
- 需明确确认风险提示

## 2. Claw Box（N100 硬件版）

### 定位

面向不想折腾的普通用户、家庭的即插即用硬件产品。

### 硬件规格

| 配置 | 标准版 | 高配版 |
|------|--------|--------|
| CPU | Intel N100（4核4线程） | Intel N100（4核4线程） |
| 内存 | 8GB DDR4 | 16GB DDR4 |
| 存储 | 128GB NVMe SSD | 256GB NVMe SSD |
| 网络 | 千兆网口 + WiFi6 | 千兆网口 + WiFi6 |
| USB | 2×USB3.0 + 2×USB2.0 | 2×USB3.0 + 2×USB2.0 |
| 视频 | HDMI 2.0 | HDMI 2.0 |
| 功耗 | 6-15W | 6-15W |
| 散热 | 无风扇/静音风扇 | 无风扇/静音风扇 |
| 尺寸 | 约 12×12×4 cm | 约 12×12×4 cm |
| **零售价** | **¥999** | **¥1499** |

### 软件预装

- **操作系统**：基于 Ubuntu Server 22.04 LTS 裁剪
- **预装软件**：
  - Claw Manager（完整功能）
  - OpenClaw Runtime（默认）
  - PicoClaw Runtime（可选切换）
- **首次启动向导**：手机扫码配置
- **OTA 更新**：自动检测并推送更新

### 开箱即用流程

```
第 1 步：开箱
├── 取出 Claw Box、电源适配器、网线
└── 检查包装清单

第 2 步：连接
├── 连接电源（LED 指示灯亮起）
├── 连接网线（或配置 WiFi）
└── 等待启动（约 30-60 秒）

第 3 步：发现设备
├── 设备屏幕/指示灯显示状态码
├── 或手机扫描设备底部二维码
└── 自动跳转到配置页面

第 4 步：初始化配置
├── 设置管理员密码
├── 选择 AI 模型（填入 API Key）
├── 配置渠道（飞书/微信/钉钉扫码授权）
└── 确认配置预览

第 5 步：开始使用
├── 自动完成 device pairing
├── Gateway 启动
└── 用户收到测试消息，确认连接成功
```

### LED 状态说明

| 状态 | 电源 LED | 状态 LED | 说明 |
|------|---------|---------|------|
| 关机 | 灭 | 灭 | 未通电 |
| 启动中 | 白（常亮） | 绿（闪烁） | 系统启动 |
| 正常运行 | 白（常亮） | 绿（常亮） | 一切正常 |
| 配置错误 | 白（常亮） | 红（闪烁） | Safe Mode |
| 网络异常 | 白（常亮） | 黄（闪烁） | 检查网络 |
| 系统更新 | 白（闪烁） | 绿（闪烁） | 正在更新 |

## 3. Claw Enterprise（企业版）

### 定位

面向小微企业的私有化 AI 助手解决方案，与 toB 配套软件打包销售。

### 功能扩展

基于 toC 内核，增加企业级功能：

| 模块 | 功能 | 说明 |
|------|------|------|
| **集中管理** | 设备管理 | 管理员可查看所有 Box 状态 |
| | 批量配置 | 统一推送配置到多台设备 |
| | 分组管理 | 按部门/项目分组管理 |
| **安全加固** | 默认安全 Runtime | 默认使用 Nanobot/IronClaw |
| | WASM 沙箱 | 技能运行在 WASM 沙箱中 |
| | 密钥管理 | 集成企业密钥管理系统 |
| **审计合规** | 操作日志 | 记录所有配置变更 |
| | 审计报表 | 导出合规报表 |
| | 数据留存 | 按企业策略留存数据 |
| **SSO 集成** | 企业微信 | 扫码登录、组织架构同步 |
| | 钉钉 | 扫码登录、组织架构同步 |
| | 飞书 | 扫码登录、组织架构同步 |
| **私有部署** | 服务器集群 | 支持 x86 服务器多实例部署 |
| | 负载均衡 | 多 Gateway 负载均衡 |
| | 高可用 | 主备模式自动切换 |

### 部署模式
