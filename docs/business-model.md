### 与 ARM 方案对比（树莓派5）

| 对比项 | 树莓派5 8GB | N100 小主机 |
|--------|------------|------------|

**结论**：N100 性价比更高，性能更强，适合作为主力方案。

## 收入模型

### toC 硬件版（Claw One）

```
收入 = 销量 × 单价

保守预估（首年）：
- 月销量：100 台
- 平均单价：¥1150
- 月营收：¥115,000
- 年营收：¥1,380,000

毛利：
- 单台毛利：约 ¥500
- 年毛利：¥600,000
```

### toB 企业版

- **不单卖 Claw**：作为 toB 配套软件的一部分
- **收费模式**：按 toB 项目整体收费，Claw 作为基础设施
- **价值**：降低 toB 客户私有化部署门槛，提升整体方案竞争力

### 免费版（Claw Desktop）价值

- **获客**：降低试用门槛，建立用户基础
- **反馈**：收集 early adopter 反馈，迭代产品
- **转化**：部分免费用户转化为硬件购买者
- **生态**：扩大 OpenClaw 社区影响力

## 成本结构

### 固定成本（月度）

| 项目 | 预估金额 |
|------|---------|
| 开发人员（2人） | ¥40,000 |
| 办公/基础设施 | ¥5,000 |
| **月度固定成本** | **¥45,000** |

### 变动成本

- 硬件 BOM：¥560/台（随销量增加）
- 售后服务：预估 5% 营收
- 物流运费：¥20-30/台

### 盈亏平衡点

```
固定成本：¥45,000/月
单台毛利：¥500

盈亏平衡销量 = 45,000 / 500 = 90 台/月

即：月销 90 台即可覆盖固定成本
```

## 竞争优势

### 与 OpenClaw 原生部署对比

| 维度 | 原生 OpenClaw | Claw One |
|------|--------------|----------|
| 部署难度 | 高（需技术背景） | 极低（即插即用） |
| 配置风险 | 高（改错即崩溃） | 低（事务配置+Safe Mode） |
| 系统隔离 | 无 | 物理隔离（硬件版） |
| 远程支持 | 困难 | 内置授权制远程 |
| 维护成本 | 高 | 低（自动更新+回滚） |

### 与云服务商托管对比

| 维度 | 云托管 | Claw One |
|------|--------|----------|
| 数据隐私 | 数据上云 | 完全本地 |
| 长期成本 | 订阅制，持续付费 | 一次性购买 |
| 网络依赖 | 依赖外网 | 局域网可用 |
| 定制化 | 受限 | 完全可控 |

## 风险与缓解

| 风险 | 影响 | 缓解策略 |
|------|------|---------|
| **N100 供应链波动** | 中 | 备选方案（N95/N200）、多供应商 |
| **OpenClaw 社区版本不兼容** | 高 | 版本锁定机制、LTS 分支、社区沟通 |
| **硬件售后成本** | 中 | 严格品控、远程诊断优先、有限保修 |
| **免费用户转化低** | 中 | 硬件差异化功能、限时优惠 |
